Font Directory for 3D Sign Creator

This directory should contain TTF or OTF font files used by the application.

Font files should be named according to the following convention:
- FontName-Style.ttf (or .otf)

Examples:
- Roboto-Regular.ttf
- OpenSans-Bold.ttf
- Lato-Italic.ttf

For the application to use these fonts, they must be listed in the fontManager.js file's initializeFontList method.

The following fonts are referenced in the application:
- Roboto-Regular.ttf
- OpenSans-Regular.ttf
- Lato-Regular.ttf
- Oswald-Regular.ttf
- Montserrat-Regular.ttf
- SourceSansPro-Regular.ttf
- Raleway-Regular.ttf
- Ubuntu-Regular.ttf
- Merriweather-Regular.ttf
- PlayfairDisplay-Regular.ttf

These fonts can be downloaded from Google Fonts or other font repositories and placed in this directory.
